# 自主執行規則

## 可以自動執行（不需要回報）
- 寫程式碼和修改程式碼
- 修復 bug 和錯誤
- 跑測試和驗證
- 建立和修改資料庫結構
- 優化效能
- 安裝套件和依賴
- 建立新檔案和資料夾
- 撰寫文件
- git add / commit / push
- 部署到 Render

## 必須停下來寫入 DECISION_NEEDED.md
- 改變系統核心架構方向
- 刪除任何現有資料或檔案
- 接觸真實金錢或 USDC 交易
- 修改智能合約邏輯
- 發現安全漏洞
- 需要外部 API Key 或新的第三方服務帳號
- 任何無法確定的重大方向決策

## 迭代原則
- 測試失敗：自動修復，最多嘗試 5 次
- 5 次後仍失敗：寫入 DECISION_NEEDED.md 並停止
- 每完成一個功能：更新 PROGRESS.md 並 git push
